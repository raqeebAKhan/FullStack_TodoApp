# FullStack_TodoApp
This website is Full Stack Todo App

## Features
* Add a new Todo
* Edit an existing todo
* Delete a todo
* complete a todo

## Technologies used
* MongoDB
* ExpressJs
* ReactJS
* NodeJS
* Redux
* Material UI
* JSON
* Axios
